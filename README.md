# Neubrutalism-landing-page

A modern landing page designed using the **Neubrutalism UI style** as part of the Front-End Development course at Route Academy.
This project focuses on applying advanced CSS concepts such as shadows, bold layouts, and strong visual hierarchy.

## 🎯 Project Objective

The goal of this assignment is to:

- Practice advanced CSS styling techniques
- Apply the Neubrutalism design trend
- Use `box-shadow` and `text-shadow` effectively
- Improve layout structuring and spacing
- Build a visually bold and trendy interface

---

## ✨ What is Neubrutalism?

Neubrutalism is a modern UI trend characterized by:

- Thick borders
- Strong drop shadows
- High contrast colors
- Bold typography
- Minimal but powerful layouts
